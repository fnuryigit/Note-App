# Note App

Basit bir **Note App** uygulaması. Kullanıcılar not ekleyebilir, renk seçebilir ve notları filtreleyebilir.

## Canlı Link
[Note App - Canlı](https://fnuryigit.github.io/Note-App/)

## Özellikler
- **Not Ekleme**: Textarea kullanarak yeni not eklenir.  
- **Renk Seçimi**: Not eklerken renk seçilebilir ve listeleme bu renge göre yapılır.  
- **Filtreleme**: Daha önce eklenmiş notlar, filtre inputu ile aranabilir.  
- **Basit ve Kullanıcı Dostu Arayüz**: HTML, CSS ve JavaScript ile geliştirilmiştir.  

## Kullanılan Teknolojiler
- HTML5  
- CSS3  
- Vanilla JavaScript  

## Kullanım
1. Textarea’ya notunuzu yazın.  
2. Renk seçici ile notunuzun rengini seçin.  
3. “Not Ekle” butonuna tıklayın.  
4. Filtreleme yapmak için üstteki inputa aramak istediğiniz kelimeyi yazın.  

## Kurulum (Opsiyonel)
1. Dosyaları bilgisayarınıza indirin: `index.html`, `style.css`, `script.js`  
2. Tarayıcıda `index.html` dosyasını açın.  

---

**Not:** Uygulama tamamen frontend tabanlıdır ve sayfa yenilenirse notlar kaybolur. LocalStorage ile geliştirilerek kaydedilebilir.
